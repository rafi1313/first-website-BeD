Selektory CSS które trzeba znać

1. *

Symbol gwiazdki wybiera wszystkie elementy na stronie.
Wielu deweloperów używa tego triku by wyzerować margin i padding.

*{
margin:0;
padding:0;
}

2. #

Symbol '#'(hash) wybiera element na stronie na podstawie id.
Wielu deweloperów używa tego triku by wyzerować margin i padding.

#container{
max-width:1200px;
margin:auto;
}
