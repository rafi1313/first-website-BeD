Selektory CSS które trzeba znać

1. *

Symbol gwiazdki wybiera wszystkie elementy na stronie.
Wielu deweloperów używa tego triku by wyzerować margin oraz padding.

*{
margin:0;
padding:0;
}

2. #X

Symbol '#'(hash) wybiera element na stronie na podstawie id.
Zgodnie z zasadą może występować tylko 1 element na stronie o danych id.

#container{
max-width:1200px;
margin:auto;
}

3. .X

Symbol '.'(kropka) jest selektorem klasy, tzn. wybiera elementy na stronie na podstawie atrybutu class.

.example{
background-color:white;
font-family: 'Source Code Pro', monospace;
}

4. X

Selektor typu jest jednym z najprostszych.
Pozwala wybierać wszystkie elementy danego typu. W przykładzie poniżej wszystkie znaczniki typu a otrzymają kolor czerwony, a wszystkie elementydiv padding równy 10px oraz biały kolor tła .

a{ color:red}
div{
padding:10px
background-color:white;
}

5. X Y

Jeden z najpopularniejszych selektów: selektor potomka.
Pozwala on na doprecyzowanie docelowego elementu, który chcemy ostylować.
W poniższym przykładzie wszystkim znacznikom a będącym bezpośrednimi potomkami elementów nav zostanie nadana wielkość tekstu 36px oraz dodane zostanie dodane podkreślenie.

nav a{
text-decoration:underline;
font-size: 36px;
}
